# Лабораторная работа по дисциплине "Бэкенд разработка веб-приложений"
### Вариант 4 - Фильмы

## Описание приложения

Приложение представляет возможность получать/создавать/обновлять (PUT/PATCH)/удалять данные по фильмам в базе данных.
Авторизация в приложении осуществляется на базе JWT токена, так же реализован досрочного отзыва токена через эндпойнт /auth/logout.

## Запуск приложения
Для запуска приложения необходим установленный Docker.

1. Необходимо заполнить переменные окружения учётными данными для БД в файле .env
```
#ip:port
DB_ADDRESS=
#database name
DB_NAME=
DB_USERNAME=
DB_PASSWORD=
```

2. Собрать образ проекта в докере выполнив команду `docker build -t delmark-backend-lab .`
3. Запустить контейнер с приложением выполнив команду `docker run -p 8080:8080 --env-file=.env delmark-backend-lab`
4. Можно тестировать приложение, url сваггера: `http://localhost:8080/swagger-ui/index.html`
